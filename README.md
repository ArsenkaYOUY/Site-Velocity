# 🚀 Velocity — Мой первый учебный проект

## 📌 О проекте

**Velocity** — это мой первый учебный проект по верстке. Я сверстал лендинг по макету из Figma, используя только **HTML, CSS и JavaScript**.

Этот проект — моя отправная точка во frontend-разработке. Здесь я впервые применил:

- Семантическую верстку
- Flexbox для построения сеток
- Организацию CSS по БЭМ (первый опыт!)
- Работу с макетом в Figma
- Табы на чистом JavaScript

---

## 🛠️ Технологии

- **HTML5** — семантическая разметка
- **CSS3** — flexbox, адаптивная верстка
- **JavaScript** — реализация табов (без библиотек!)
- **БЭМ** — методология именования классов

---

## ✨ Что умеет проект

- ✅ Полностью сверстанный лендинг по макету
- ✅ Адаптивная верстка (корректно отображается на разных устройствах)
- ✅ Интерактивные табы (переключение контента без перезагрузки)
- ✅ Чистый код без фреймворков

## 🚦 Запуск проекта

1. Скачайте или клонируйте репозиторий:
```bash
git clone https://github.com/ArsenkaYOUY/site_velocity.git
```

2. Откройте `index.html` в любом браузере.

Никаких дополнительных установок не требуется — это чистый HTML/CSS/JS! 🎉
